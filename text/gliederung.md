#Gliederung:


##Workflow:

###Aufsetzen eines Projektes:
- Anlegen git Repo
- Anlegen DB
- Selecting CMS
- Downloading necessary data
- Editing config -> add database name
- Run setup to install the cms
- Setup local server
- Git push initial commit

###During development:
- watch prozesse
- Präprozessoren
- Minifizierung Javascript, uglify und co
- Bildanpassungen
- Fehlertesting
- Livetesting

###Deployment:
- Build Prozess
- Copy files via ftp/sftp to server
- Database deployment
	- Database diff (UUID)
	- Database upload
- Testing, Staging, Production
